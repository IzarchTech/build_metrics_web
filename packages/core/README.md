BuildMetric Core Library

## Getting Started

Build library:

```bash
npm run build
# or
pnpm build
# or
yarn build
# or
bun build
```

Run lint:

```bash
npm run lint
# or
pnpm lint
# or
yarn lint
# or
bun lint
```
